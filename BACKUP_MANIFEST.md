## Restore Instructions

### To restore the site from scratch:
```bash
git clone https://github.com/yachtyhermes-bit/the-signal.git
cd the-signal
npm install
node build.js
vercel --prod --token $VERCEL_TOKEN
```

### To restore cron jobs, use Hermes:
```bash
hermes cron create --prompt "..." --schedule "..." --workdir /home/chino/thesignal
```
Refer to the tables above for the exact schedules and configurations.

### Data files are regenerated by:
```bash
node scripts/fetch-prices.js        # → data/prices.json
python3 scripts/refresh-scorecard.py  # → data/scorecard.json
python3 scripts/refresh-highlights.py # → data/highlights.json
```
