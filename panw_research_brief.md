## 4. KEY PRODUCTS & PLATFORMS

### Strata — Network Security (NGFW)
- Next-generation firewalls (traditional core business)
- Prisma SASE (Secure Access Service Edge) — cloud-delivered network security
- Includes SD-WAN (from CloudGenix acquisition)
- **Competitive displacement:** Prisma Access winning deals vs. Zscaler, Netskope

### Prisma — Cloud Security
- **Prisma Cloud:** CNAPP (Cloud-Native Application Protection Platform)
- Covers cloud workload protection, posture management, container security, IaC security
- **Prisma SASE:** Combines SD-WAN with cloud-delivered security
- **Prisma AIRS 3.0:** AI Runtime Security — protects AI agents in runtime (launched 2026)
- **Prisma Browser for Business:** Secure enterprise browser (launched 2026)

### Cortex — Security Operations (SOC)
- **Cortex XSIAM:** AI-driven security operations platform (flagship SOC product)
- Cortex XDR: Extended detection and response
- **AgentiX:** Agentic SOC capabilities
- Big win: IBM QRadar SaaS asset acquisition (~$1.14B, Sep 2024) — transitioning QRadar customers to XSIAM

### Idira — Identity Security (NEW, May 2026)
- Next-generation identity security platform built from the **CyberArk acquisition** ($25B, closed FY2026)
- Modern privileged access management for human, machine, and **agentic AI identities**
- Addresses 109:1 machine-to-human identity ratio
- Launched at IMPACT 2026 conference

### Unit 42 — Threat Intelligence & Consulting
- Threat research, incident response, MDR services
- Publishes regular threat intelligence reports

## 6. COMPETITIVE POSITION

### vs. CrowdStrike (CRWD)
- **PANW Advantage:** Broader platform (network + cloud + SOC + identity); $265B vs CRWD ~$80-100B market cap; platform consolidation story
- **CRWD Advantage:** Higher revenue growth rate historically (41% CAGR FY21-FY26 vs PANW 22% CAGR FY20-FY25); Pure-play endpoint/XDR leader
- **Dynamic:** PANW winning via platform consolidation; CRWD winning via endpoint dominance and Falcon platform

### vs. Fortinet (FTNT)
- **PANW Advantage:** Cloud and AI security capabilities; higher revenue growth; enterprise focus
- **FTNT Advantage:** Stronger in mid-market and SMB; higher operating margins; more cost-effective NGFW

### vs. Check Point (CHKP)
- **PANW Advantage:** Modern cloud-native architecture; AI-driven SOC; platform breadth; 10x+ revenue growth rate
- **CHKP Advantage:** Higher profitability; long-standing enterprise relationships; strong firewall installed base

### vs. Zscaler (ZS)
- **PANW Advantage:** Prisma SASE competing directly; broader platform; better margins
- **ZS Advantage:** Pure-play SASE/Zero Trust; higher growth historically

### CEO Nikesh Arora's Take (Q3 Call):
